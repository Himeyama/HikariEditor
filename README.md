# ひかりエディタ

![image](https://github.com/Himeyama/HikariEditor/assets/39254183/2d1c6ef6-0f09-4492-bb13-08f815242f73)

Windows 向けのモダンなテキスト／コードエディタです。WinUI 3 と Monaco Editor を組み合わせ、VSCode ライクな編集体験を提供します。

## 機能

- **マルチタブ編集** — TabView で複数ファイルを同時に開いて編集できます
- **Monaco Editor** — VSCode と同じエンジンによるシンタックスハイライトとコード補完
- **ファイルエクスプローラー** — TreeView でディレクトリ構造を表示し、ファイルの作成・削除・切り替えが可能
- **自動保存** — タイトルバーのトグルで有効／無効を切り替えられます
- **LaTeX サポート** — `.tex` ファイルをコンパイルし、生成された PDF をインライン プレビュー表示
- **ターミナル** — 複数のターミナルタブを管理できます
- **改行コード切り替え** — ステータスバーから LF／CRLF をワンクリックで切り替え
- **Mica バックドロップ** — Windows 11 のモダンなアクリル素材 UI

## 動作環境

| 項目 | 要件 |
|------|------|
| OS | Windows 10 22H2 (build 22621) 以上 |
| アーキテクチャ | x64 |
| ランタイム | 不要（Self-Contained） |

## キーボードショートカット

| キー | 操作 |
|------|------|
| `Ctrl+O` | ディレクトリを開く |
| `Ctrl+S` | ファイルを保存 |
| `Ctrl+C` | コピー |
| `Ctrl+V` | ペースト |
| `Ctrl+T` | ターミナルを開く |
| `Ctrl+U` | ログを開く |
